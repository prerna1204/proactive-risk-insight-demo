## Project Overview

Proactive Risk Insight is an AI-powered financial risk monitoring system designed to detect **early signs of financial distress in banking customers before they miss loan payments**.

Banks usually act only **after customers default**, which increases financial loss and damages customer relationships.

This system enables **early intervention** by predicting the probability of loan default using behavioral and financial indicators.

The platform analyzes customer financial patterns and classifies them into **Low Risk, Medium Risk, or High Risk**, allowing banks to take preventive action.
